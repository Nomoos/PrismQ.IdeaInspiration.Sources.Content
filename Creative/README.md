# Creative Source Module

**Creative resources and inspiration for content creation**

## Overview

This module aggregates creative resources from multiple sources - creative cues and inspiration including lyric snippets, narrative structures, and visual aesthetics. Based on the PrismQ Source Taxonomy category **Creative**.

## Source Category

**Category**: Creative  
**Purpose**: Creative resources and inspiration  
**Value**: Spark creativity and provide artistic direction

## Sources

This source collects data from the following creative sources:

- **LyricSnippetsSource** - Lyric fragments and poetic phrases
- **ScriptBeatsSource** - Narrative structures and story beats
- **VisualMoodboardSource** - Visual aesthetics and mood inspiration

- NVIDIA GPU with CUDA support (optional, for future AI features)
